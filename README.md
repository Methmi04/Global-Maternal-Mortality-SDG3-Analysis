# Global-Maternal-Mortality-SDG3-Analysis


## Overview
This project investigates maternal mortality trends worldwide through advanced **SQL data preparation** and **Power BI dashboards**. It supports evidence-based monitoring of **SDG 3.1.1 (Maternal Mortality Ratio)**, highlighting disparities across countries and regions from 2005–2023.

## Key Features
- **SQL Server Data Preparation**
  - Database creation and inspection
  - Unpivoting year columns for time-series analysis
  - Removing redundant/irrelevant fields
  - Standardizing text fields
  - Identifying and removing duplicates
  - Creating a cleaned dataset (`SDG_Cleaned`) for Power BI

- **Power BI Modeling & DAX**
  - Year-over-Year (YoY) Change (%)
  - Rank by Value
  - Progress vs Target (%)
  - Improvement Since 2015 (%)
  - Average Indicator Value
  - Top 5 Countries Classification
  - 3-Year Rolling Average

- **Dashboard Visualizations**
  - Waterfall Chart (progress over time)
  - Filled Map (regional disparities)
  - Heatmap (country-year comparison)
  - Gauge Chart (progress vs target)
  - Line Chart, Pie Chart, Cards, and Slicers

## Dataset
- Source: **United Nations SDG Data Portal**
- Coverage: 196 countries, 2005–2023
- Indicator: Maternal Mortality Ratio (per 100,000 live births)

## Insights
- Significant global reduction in maternal mortality since 2005
- Persistent disparities between regions (e.g., Sub-Saharan Africa vs Europe)
- Identification of top-performing countries since 2015
- Clear visualization of progress toward the 2030 target
